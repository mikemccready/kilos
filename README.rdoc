KILOS is a training resource for combat sport athletes. The app allows users to follow their favorite members and share useful techniques through uploaded video. 

Notable features:

Heroku deployed app: https://kilosbjj.herokuapp.com/
CRUD functionality with video and user models
Fully responsive design
User profile creation
User authentication
Paperclip for image upload
Video posting
Follow Users
Asset Pipeline

Stack:

HTML5 & CSS3
Bootstrap
Javascript & jQuery
Ruby on Rails
Heroku
